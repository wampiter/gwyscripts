Batch image processing scripts for gwyddion

Use batch_merge_proc to take a whole folder of park files and turn into .gwy files consolidated by scan and processed by channel, as well as pngs

image_stitch can be used to stich together multiple scans arranged in a line, and also multiple channels of a scan or set of scans

save_proc is a quick script to save-as open files to a proc folder and automatically generate pngs for them

Finally, rescale_units simply has some useful gwy functions for this particular use case.
